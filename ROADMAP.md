# PrepVerse Project Roadmap

This roadmap outlines the past, present, and future direction of PrepVerse. Our goal is to empower students and career changers worldwide with a completely free, community-curated, and AI-driven placement preparation ecosystem.

---

## 🗺️ Phased Milestones

### 📍 Phase 1: Core Foundation (v1.0.0 - Current)
*Focus: Establish the primary functional modules, curriculum registries, and AI evaluation loops.*
- [x] **Universal Placement Dashboard**: Real-time progress trackers, gamified daily sheets, and activity streak analytics.
- [x] **Interactive DSA Module**: Multi-sheet curriculum list (Blind 75, etc.) with functional client-side code playgrounds.
- [x] **Resume Studio**: Direct resume analysis and feedback loops using local or AI systems.
- [x] **Interactive Interview Hub**: Voice/text simulation of core behavioral and technical interview questions.
- [x] **Aptitude Evaluation**: Time-boxed quantitative and logical puzzles with automated scoring.
- [x] **Administrative Moderation Panel**: Real-time management console for problem curricula and candidate registration logs.
- [x] **Open Source Hub**: Contributor guides, bounty trackers, and permissive licensing structures.

---

### 📍 Phase 2: Collaboration & Social (v2.0.0 - Coming Q3 2026)
*Focus: Peer-to-peer engagement, persistent multiplayer challenges, and resume revision forums.*
- [ ] **Durable Cloud Persistence**: Fully integrate Firebase Firestore to persist candidate stats, code solutions, custom plans, and history.
- [ ] **Collaborative Mock Coding**: Real-time coding rooms using WebSockets for peer-to-peer interviews and whiteboard reviews.
- [ ] **Community Threads**: Interactive forums with categorization, post upvoting, replies, and "Accept Answer" status for Q&As.
- [ ] **Multiplayer DSA Duels**: Speed-coding matches with real-time progress indicators of opponent's test case passes.
- [ ] **Integrated Calendar Alerts**: Sync study schedules directly to Google Calendar using Workspace API integrations.

---

### 📍 Phase 3: Premium AI Coaching & Integrations (v3.0.0 - Coming Q1 2027)
*Focus: State-of-the-art AI code grading, resume PDF parsing, and customized syllabus generators.*
- [ ] **Gemini Code Evaluation Engine**: Advanced, server-side code analysis using the `@google/genai` SDK to evaluate space/time complexity and code safety.
- [ ] **Automated PDF Resume Parsing**: Deep resume analysis by extracting content directly from PDF uploads to generate an interactive improvement checklist.
- [ ] **Smart Mock Interviews**: Audio-first mock interviews leveraging the Google Gen AI Live API for low-latency conversational interviews.
- [ ] **Personalized Syllabus Generation**: Generate a customized DSA study plan using the Gemini API based on the candidate's target company and weak areas.

---

### 📍 Phase 4: Enterprise Placement Pipelines (v4.0.0 - Coming Q4 2027)
*Focus: University recruiter dashboards, automated screening systems, and certified placement pipelines.*
- [ ] **University Admin Dashboard**: Allow college placement departments to register students, monitor real-time streaks, and issue mock assessments.
- [ ] **Recruiter Talent Finder**: Permission-based portal for company recruiters to search candidates by verified DSA solved counts and resume rankings.
- [ ] **Placement Drives Integration**: Automated end-to-end coding assessments with browser-lock mechanisms and automated cheating flags.
- [ ] **Global Leaderboards**: Country-specific and university-specific competitive leaderboards to increase engagement and motivation.
